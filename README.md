# 📱 Realidade Aumentada com Marcador Hiro

Este projeto demonstra a utilização de **Realidade Aumentada (AR)** baseada em marcadores, utilizando o clássico **marcador Hiro** para sobrepor objetos virtuais no mundo real.

---

## 🧠 O que é Realidade Aumentada?

A **Realidade Aumentada (AR)** é uma tecnologia que permite a sobreposição de elementos virtuais (como modelos 3D, imagens ou animações) sobre o ambiente físico, em tempo real, através de uma câmera.

---

## 🟦 Sobre o Marcador Hiro

O **Hiro Marker** é um dos marcadores mais conhecidos em aplicações de AR. Ele é amplamente utilizado em bibliotecas como AR.js e ARToolKit.

### 🔍 Características:

* Fácil detecção por câmera
* Alto contraste (preto e branco)
* Muito usado para testes e protótipos
* Identificado por padrão como `"hiro"`

---

## 🚀 Como Acessar

O projeto está disponível online via **GitHub Pages**, não sendo necessário instalar nada.

### 👉 Passos:

1. Acesse o link do projeto:

```
 https://marcos-cavalaro.github.io/Sensor_Temperatura/
```

2. Permita o acesso à câmera quando solicitado

3. Aponte a câmera para o **marcador Hiro**

4. O objeto 3D será exibido automaticamente sobre o marcador

---

## 📷 Como Usar o Marcador Hiro

* Você pode **imprimir o marcador Hiro** ou exibi-lo em outra tela
* Certifique-se de que o marcador esteja bem iluminado
* Mantenha a câmera estável para melhor detecção
